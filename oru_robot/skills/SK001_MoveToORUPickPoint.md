
# Skill: MoveToORUPickPoint (移动到ORU抓取点)

## 描述
移动到ORU抓取点，收拢夹爪抓取ORU

## 输入参数
| 参数名 | 类型 | 必需 | 描述 |
|--------|------|------|------|
| oru_id | str | 是 | ORU物体ID |
| approach_height | float | 否 | 接近高度（安全高度） |

## 输出
- 成功: True
- 失败: False

## 执行步骤
1. 从ObjectGraph获取ORU抓取点位置
2. 移动到安全高度
3. 下降到抓取位置
4. 收拢夹爪
5. 确认抓取成功

## 前置条件
- 机器人在安全位置
- 夹爪处于松开状态
- ORU物体可访问

## 后置条件
- 机器人在ORU抓取位置
- 夹爪收拢并抓取ORU

## 错误处理
- 位置不可达
- 抓取失败
- 安全传感器触发
