
# Skill: PickScrewdriver (抓取螺丝刀工具)

## 描述
抓取螺丝刀工具

## 输入参数
| 参数名 | 类型 | 必需 | 描述 |
|--------|------|------|------|
| tool_rack_id | str | 是 | 工具架ID |
| screwdriver_id | str | 是 | 螺丝刀ID |

## 输出
- 成功: True
- 失败: False

## 执行步骤
1. 获取螺丝刀位置
2. 移动到安全高度
3. 移动到抓取位置
4. 收拢夹爪
5. 确认抓取
6. 上升到安全高度

## 前置条件
- 夹爪松开
- 螺丝刀在工具架上

## 后置条件
- 螺丝刀已抓取

## 错误处理
- 螺丝刀不在位
- 抓取失败
