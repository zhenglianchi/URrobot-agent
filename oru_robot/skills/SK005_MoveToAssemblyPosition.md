
# Skill: MoveToAssemblyPosition (移动到装配位置)

## 描述
移动到装配位置

## 输入参数
| 参数名 | 类型 | 必需 | 描述 |
|--------|------|------|------|
| assembly_station_id | str | 是 | 装配工位ID |

## 输出
- 成功: True
- 失败: False

## 执行步骤
1. 获取装配工位位置
2. 移动到安全高度
3. 移动到装配位置上方
4. 确认位置

## 前置条件
- ORU已抓取
- 装配工位可用

## 后置条件
- 机器人在装配位置上方

## 错误处理
- 装配工位被占用
- 路径不可达
