
# Skill: InsertORU (进行装配插入)

## 描述
进行装配插入（带力控）

## 输入参数
| 参数名 | 类型 | 必需 | 描述 |
|--------|------|------|------|
| insertion_depth | float | 是 | 插入深度 |
| max_force | float | 是 | 最大力阈值 |
| speed | float | 否 | 插入速度 |

## 输出
- 成功: True
- 失败: False

## 执行步骤
1. 确认在装配位置上方
2. 以低速开始插入
3. 监控力反馈
4. 到达插入深度后停止
5. 松开夹爪
6. 上升到安全高度

## 前置条件
- 在装配位置上方
- ORU已抓取

## 后置条件
- ORU装配完成
- 夹爪松开

## 错误处理
- 力超限
- 插入深度未达到
- 位置偏差
