
# Skill: MoveToStorageRack (移动到收纳架)

## 描述
移动到收纳架，松开夹爪放置ORU

## 输入参数
| 参数名 | 类型 | 必需 | 描述 |
|--------|------|------|------|
| storage_rack_id | str | 是 | 收纳架ID |
| slot_id | str | 否 | 槽位ID |

## 输出
- 成功: True
- 失败: False

## 执行步骤
1. 获取收纳架放置位置
2. 移动到安全高度
3. 移动到放置位置上方
4. 下降到放置位置
5. 松开夹爪
6. 上升到安全高度

## 前置条件
- ORU已抓取
- 收纳架槽位为空

## 后置条件
- ORU放置在收纳架上
- 夹爪松开
- 机器人在安全位置

## 错误处理
- 槽位被占用
- 放置位置不可达
