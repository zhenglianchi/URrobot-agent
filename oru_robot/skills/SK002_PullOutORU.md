
# Skill: PullOutORU (向上移动拔出ORU)

## 描述
向上移动拔出ORU

## 输入参数
| 参数名 | 类型 | 必需 | 描述 |
|--------|------|------|------|
| pull_height | float | 是 | 拔出高度 |
| speed | float | 否 | 移动速度 |

## 输出
- 成功: True
- 失败: False

## 执行步骤
1. 确认ORU已抓取
2. 以设定速度向上移动
3. 到达拔出高度后停止
4. 确认位置

## 前置条件
- ORU已被抓取
- 夹爪保持收拢状态

## 后置条件
- ORU完全拔出
- 机器人在安全高度

## 错误处理
- 力超限
- 位置偏差过大
