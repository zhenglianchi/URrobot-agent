
# Skill: PlaceScrewdriver (放置螺丝刀工具)

## 描述
放置螺丝刀工具

## 输入参数
| 参数名 | 类型 | 必需 | 描述 |
|--------|------|------|------|
| tool_rack_id | str | 是 | 工具架ID |
| screwdriver_id | str | 是 | 螺丝刀ID |

## 输出
- 成功: True
- 失败: False

## 执行步骤
1. 移动到工具架安全高度
2. 移动到放置位置上方
3. 下降到放置位置
4. 松开夹爪
5. 上升到安全高度

## 前置条件
- 螺丝刀已抓取
- 工具架槽位为空

## 后置条件
- 螺丝刀放置在工具架上
- 夹爪松开

## 错误处理
- 槽位被占用
- 放置失败
