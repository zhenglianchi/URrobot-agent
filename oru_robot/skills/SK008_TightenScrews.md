
# Skill: TightenScrews (依次旋拧ORU螺丝)

## 描述
依次旋拧ORU螺丝

## 输入参数
| 参数名 | 类型 | 必需 | 描述 |
|--------|------|------|------|
| screw_positions | list[list[float]] | 是 | 螺丝位置列表 |
| target_torque | float | 是 | 目标扭矩 |

## 输出
- 成功: True
- 失败: False

## 执行步骤
1. 遍历每个螺丝位置
2. 移动到螺丝位置上方
3. 下降接触螺丝
4. 启动螺丝刀
5. 监控扭矩
6. 达到目标扭矩后停止
7. 移动到下一个螺丝

## 前置条件
- 螺丝刀已抓取
- ORU已装配

## 后置条件
- 所有螺丝拧紧到目标扭矩

## 错误处理
- 扭矩超限
- 螺丝滑牙
- 位置偏差
