
# Skill: PickNewORU (抓取新的ORU)

## 描述
抓取新的ORU

## 输入参数
| 参数名 | 类型 | 必需 | 描述 |
|--------|------|------|------|
| storage_rack_id | str | 是 | 收纳架ID |
| slot_id | str | 是 | ORU所在槽位ID |

## 输出
- 成功: True
- 失败: False

## 执行步骤
1. 移动到收纳架安全高度
2. 下降到ORU位置
3. 收拢夹爪
4. 向上拔出
5. 确认抓取

## 前置条件
- 槽位有ORU
- 夹爪处于松开状态

## 后置条件
- 新ORU已抓取
- 机器人在安全高度

## 错误处理
- 槽位为空
- 抓取失败
